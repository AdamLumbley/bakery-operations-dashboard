# Bakery Operations Dashboard

![Bakery Operations Dashboard](https://raw.githubusercontent.com/AdamLumbley/bakery-operations-dashboard/main/Bakery%20Operations%20Dashboard.png)

## Overview

The Bakery Operations Dashboard is a business intelligence project designed to provide operational visibility into bakery performance metrics, production trends, inventory movement, staffing efficiency, and sales insights.

Built as a modern analytics dashboard, this project demonstrates how data visualization can help bakery owners and operations teams make faster, data-driven decisions.

---

## Features

* 📊 Executive-level KPI tracking
* 🥐 Product sales performance analysis
* 📦 Inventory and supply monitoring
* 👩‍🍳 Staffing and labor insights
* 📈 Trend analysis and forecasting visuals
* 🕒 Daily operational performance tracking
* 🎯 Clean and interactive dashboard design

---

## Dashboard Preview

The dashboard includes:

* Revenue and sales KPIs
* Production efficiency metrics
* Product category breakdowns
* Inventory status indicators
* Time-based trend analysis
* Operational summaries for quick decision-making
